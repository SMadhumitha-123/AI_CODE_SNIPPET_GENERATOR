## Method 1: Using String Slicing (Most Pythonic)

```python
def reverse_string_slicing(text):
    """
    Reverse a string using Python's slicing feature.
    This is the most common and efficient method.
    """
    return text[::-1]

# Example usage
original = "Hello World"
reversed_str = reverse_string_slicing(original)
print(f"Original: {original}")
print(f"Reversed: {reversed_str}")
```

## Method 2: Using Built-in `reversed()` Function

```python
def reverse_string_builtin(text):
    """
    Reverse a string using the built-in reversed() function.
    reversed() returns an iterator, so we join it back to a string.
    """
    return ''.join(reversed(text))

# Example usage
original = "Python Programming"
reversed_str = reverse_string_builtin(original)
print(f"Original: {original}")
print(f"Reversed: {reversed_str}")
```

## Method 3: Using a Loop (Manual Approach)

```python
def reverse_string_loop(text):
    """
    Reverse a string using a for loop.
    This method manually builds the reversed string character by character.
    """
    reversed_str = ""
    for char in text:
        reversed_str = char + reversed_str
    return reversed_str

# Example usage
original = "Learning Python"
reversed_str = reverse_string_loop(original)
print(f"Original: {original}")
print(f"Reversed: {reversed_str}")
```

## Method 4: Using Recursion

```python
def reverse_string_recursive(text):
    """
    Reverse a string using recursion.
    Base case: if string is empty or has one character, return it.
    Recursive case: return last character + reverse of remaining string.
    """
    # Base case
    if len(text) <= 1:
        return text
    
    # Recursive case
    return text[-1] + reverse_string_recursive(text[:-1])

# Example usage
original = "Recursion"
reversed_str = reverse_string_recursive(original)
print(f"Original: {original}")
print(f"Reversed: {reversed_str}")
```

## Explanation of Each Method:

1. **String Slicing (`[::-1]`)**: 
   - Most efficient and Pythonic
   - Uses Python's slice notation with step -1 to traverse backwards
   - Time complexity: O(n), Space complexity: O(n)

2. **`reversed()` Function**:
   - Uses Python's built-in function that returns a reverse iterator
   - Requires `join()` to convert back to string
   - Time complexity: O(n), Space complexity: O(n)

3. **Loop Method**:
   - Manual approach that builds string character by character
   - Good for understanding the process
   - Time complexity: O(n²) due to string concatenation, Space complexity: O(n)

4. **Recursion**:
   - Elegant but not efficient for large strings
   - Can cause stack overflow for very long strings
   - Time complexity: O(n), Space complexity: O(n) due to call stack

**Recommendation**: Use the slicing method (`text[::-1]`) for most cases as it's the most readable and efficient approach in Python.
